# Playwright E2E Testing Framework

## Overview
This project demonstrates end-to-end automation testing using Playwright with TypeScript.

## Features
- Login testing (valid & invalid)
- Add to cart functionality
- Checkout flow
- Page Object Model (POM)

## Tech Stack
- Playwright
- TypeScript
- Node.js

## How to Run
1. Install dependencies:
   npm install

2. Run tests:
   npx playwright test
